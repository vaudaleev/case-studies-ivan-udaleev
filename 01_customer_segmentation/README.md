# Сегментация клиентов «Золотая жила»

## Задача
Определить наиболее прибыльный клиентский сегмент для оптимизации маркетингового бюджета.  
CAC = ₽15 000, план привлечения = 1 000 клиентов.

## Данные
- 8 712 записей
- Признаки: `Age`, `Gender`, `Profit`
- Изначально: средняя прибыль ₽15 755/клиент (высокий CAC съедал маржу)

## Подход
1. **Очистка**: выявление и удаление выбросов через Boxplot + IQR-метод
2. **EDA**: сравнение profitability по полу и возрастным группам
3. **Глубокая сегментация**: разделение широкой категории `18–24` на `18–20` и `21–24`
4. **Валидация**: проверка стабильности метрик на подвыборках

## Результаты
| Сегмент | Ср. Profit/клиент | Вывод |
|---------|-------------------|-------|
| Женщины 18–24 | ₽26 278 | Выглядел лидером на поверхностном анализе |
| Мужчины 18–20 | ₽2 042 | Тянул вниз средний показатель всей группы |
| **Мужчины 21–24** | **₽79 333** | ✅ Реальная «золотая жила» (+6x к базе) |

## Бизнес-рекомендации
- Сместить фокус таргетинга на `Мужчины 21–24`
- Разделить маркетинговые бюджеты: широкая аудитория `18–24` → ретаргетинг на `21–24`
- Пересчитать LTV/CAC для нового сегмента перед масштабированием
